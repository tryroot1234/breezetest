# Changelog

All notable changes to BreezeTest will be documented in this file.

## [0.1.0] - 2026-05-18

### Added
- Initial release
- YAML-driven test definitions
- 30+ built-in browser actions (navigation, interaction, assertions, waits, capture)
- Data-driven testing with inline data and CSV support
- Variable system with `{{var}}` syntax
- Self-contained HTML report with embedded screenshots
- JUnit XML output for CI integration
- Rich console output
- Parallel test execution
- CLI with `run`, `init`, `list`, `report` commands
- Plugin system for extensibility
- GitHub Actions workflow templates
